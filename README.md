# Public REST API Integration & Dynamic Analytics Dashboard Workspace

A progressive frontend engineering case study tracking the development of an asynchronous data system from a simple API fetch engine into a high-performance, fault-tolerant analytics application. 

## 🌐 Live Production Deployments

* **🚀 Milestone 3 (Enterprise Optimized Production Engine):** [Launch Live Dashboard v3](https://kyngpyng.github.io/API-Integration/v3-optimized/index.html)
* **📊 Milestone 2 (Data Visualization Dashboard):** [Launch Analytics v2](https://kyngpyng.github.io/API-Integration/v2-dashboard/index.html)
* **📄 Milestone 1 (Original Standard Core Fetch):** [Launch Core App v1](https://kyngpyng.github.io/API-Integration/index.html)
